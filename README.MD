# Telco Customer Churn Analysis

This project analyzes telecom customer churn using Python, SQL, and PostgreSQL.

## Tools Used
- Python (Pandas, Matplotlib)
- SQL
- PostgreSQL
- pgAdmin

## Project Steps
1. Data Cleaning
2. Data Exploration
3. SQL Analysis
4. Customer Segmentation
5. Churn Insights

## Key Insights
- Customers with month-to-month contracts churn more
- High monthly charges increase churn probability
- Customers with short tenure churn frequently
